=== Comment Word Count ===Contributors: nickmomrikTags: comments, count, wordsStable tag: 2.0Tested up to: 4.4Outputs the total number of words in all comments.== Installation ==1. Upload `comment-word-count.php` to the `/wp-content/plugins/` directory1. Activate the plugin through the 'Plugins' menu in WordPress1. Place `<?php mdv_comment_word_count(); ?>` in your templates.
